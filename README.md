# JKalman Processing Library

JKalman is a Kalman Filter library written by Petr Chmelar.

For the original source code see:
https://sourceforge.net/projects/jkalman/

## Installation

Copy this directory into `processing/libraries`

## Build

```bash
gradle -q
```

## Generate JavaDoc

```bash
gradle javadoc
```
